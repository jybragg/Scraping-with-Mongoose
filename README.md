# WebDev_News

A web app that lets users view and leave comments on the latest web development articles scraped from Reddit.com

## Development and Technical Details
### This app uses:

Node.js, Express.js
<br> 
MongoDB
<br>
Cheerio
<br>
Handlebars, Bootstrap, CSS
